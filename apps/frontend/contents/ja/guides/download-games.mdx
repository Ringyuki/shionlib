## ダウンロードパネルを開く

任意のゲームエントリーページで、上部の **ダウンロード** ボタンをクリックするとダウンロードパネルが開きます。

パネルにはそのゲームの利用可能なリソースが一覧表示され、ダウンロード数の多い順に並んでいます。

> すべてのリソースはコミュニティユーザーによってアップロードされています。ファイルの安全性はご自身でご確認ください。

## リソースカードのタグを読む

各リソースカードの上部には、そのリソースの対応範囲を説明するタグが表示されます。

### プラットフォーム

**Windows**、**Android**、**iOS**、**PlayStation Vita** など、リソースが動作するプラットフォームを示します。

複数のプラットフォームが表示される場合は、アーカイブに複数プラットフォーム向けのビルドが含まれているか、クロスプラットフォームで動作することを意味します。

### 言語

ゲームまたはパッチの言語を示します：

| タグ     | 意味                             |
| -------- | -------------------------------- |
| 日本語   | 日本語（原版または日本語パッチ） |
| 简体中文 | 中国語簡体字ローカライズ         |
| 繁體中文 | 中国語繁体字ローカライズ         |
| English  | 英語版または英語ローカライズ     |
| その他   | その他の言語                     |

### エミュレーターエンジン（Android / iOS のみ）

プラットフォームが Android または iOS の場合のみ表示されます。リソースで使用されているゲームエンジンを示し、適切なエミュレーターの選択を助けます：

| タグ            | エンジン                                |
| --------------- | --------------------------------------- |
| KiriKiri (KRKR) | KiriKiri / KRKRZ エンジン               |
| ONScripter      | ONScripter エンジン                     |
| Artemis         | Artemis Engine                          |
| その他エンジン  | その他のエンジン — リソースのメモを参照 |

各エンジンに対応するエミュレーターについては、[主要エンジンとエミュレーター対応表](/ja/docs/guides/common-vn-engines-and-emulators) を参照してください。

### メモ

一部のリソースにはアップロード者が記入したメモがあります（ファイルエントリーの下に表示）。バージョン情報、含まれるコンテンツ、または特別な注意事項が記載されています。ダウンロード前に確認することをお勧めします。

## ファイルエントリーの詳細

各リソースカードには 1 つ以上のファイルエントリーが含まれる場合があり、各エントリーには以下が表示されます：

- **ファイル名とサイズ** — 正しいファイルをダウンロードしているか確認
- **ファイルステータス** — 下記参照
- **ハッシュ値** — ダウンロード後のファイル整合性の確認に使用

### ファイルステータス

| ステータス             | 意味                                                          |
| ---------------------- | ------------------------------------------------------------- |
| S3（緑）               | ファイルは準備完了、ダウンロード可能                          |
| 処理中（黄）           | ファイルをスキャン・アップロード中 — 後でリロードしてください |
| アップロード待ち       | ファイルは処理キューに入っています                            |
| アップロード者のみ表示 | ファイルの処理が完了しておらず、アップロード者にのみ表示      |

**S3** ステータスのファイルのみダウンロードできます。

## ファイルをダウンロードする

ファイルエントリー右側のボタンをクリックしてダウンロードを開始します。リンクが生成される前に **Cloudflare Turnstile** チャレンジ（ボット認証）が表示されます。通過後にリンクが生成されます。

> ダウンロードリンクには有効期限があります：10 GB 未満のファイルは **120 分**、10〜20 GB のファイルは **240 分**、20 GB 超のファイルは **360 分** です。期限内にダウンロードを完了してください。

### ブラウザダウンロード

**地球アイコン** ボタンをクリックすると、ブラウザで直接ダウンロードできます。ほとんどのユーザーに適しています。

### Aria2 へのプッシュ

**稲妻アイコン** ボタンをクリックすると、ローカルまたはリモートで動作している Aria2 ダウンロードマネージャーにダウンロードリンクをプッシュします。

使用前に **アカウント設定 → ダウンロード設定** で Aria2 の接続情報（プロトコル、ホスト、ポート、シークレット、ダウンロードパス）を設定してください。接続に失敗した場合はエラーが表示され、設定ページに案内されます。

## ファイルハッシュを検証する

ダウンロード後、ハッシュ値を比較してファイルの整合性を確認することをお勧めします：

1. ファイルエントリーに表示されているハッシュ値をコピー（コピーボタンをクリック）
2. システムツールまたはサードパーティソフトを使用してダウンロードファイルのハッシュを計算
3. 2 つの値を比較 — 完全に一致していればファイルは正常

Shionlib は現在 **BLAKE3** と **SHA-256** の 2 つのハッシュアルゴリズムをサポートしています。使用するアルゴリズムはファイルエントリーに明記されています。

## 誤検知ウイルスフラグ

ファイルエントリーに赤い **「ウイルスの可能性あり」** タグが表示されている場合、自動スキャンで以前にウイルスの疑いが検出されたものの、管理者がレビューして**誤検知**と確認し、手動で公開したことを意味します。

タグにカーソルを合わせると検出されたウイルス名が表示されます。

> ビジュアルノベルゲーム（特に KiriKiri などのエンジンを使用する作品）は、パッケージング方式の特性上、**ウイルス対策ソフトに誤検知されやすい** です。ソースが信頼できる場合は安全にダウンロードできます。

## 更新履歴

リソースカード右下の **···** メニューをクリックして **更新履歴** を選択すると、そのリソースの過去のバージョンをすべて確認できます。ファイルサイズ、ハッシュ、更新日時、アップロード者が記入した更新理由が含まれます。

## リソースを報告する

リソースに規約違反のコンテンツ（マルウェア、著作権問題など）が含まれていると判断した場合は、**···** メニューをクリックして **違反を報告** を選択し、理由を記入してください。悪意のある報告は適切に対処されます。
