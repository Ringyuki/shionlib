## はじめに

このガイドは、「ダウンロードしたゲームをどのエミュレーター/ランタイムで起動すべきか」を素早く判断するためのものです。

基本ルールは 2 つです。

- Windows 実行ファイルなどのネイティブ版があるなら、まずネイティブ実行を優先する。
- エミュレーターが必要な場合は、公開ソースがあり継続メンテナンスされているプロジェクトを優先する。

## 早見対応

Android / iOS リソースには **シミュレーターエンジン** のラベルが付きます。  
現在のラベル値は `KRKR` / `ONS` / `ARTEMIS` / `OTHER` です。

このラベルはリソースのエンジン種別を示すもので、「必ずこの 1 つのアプリだけを使う」という意味ではありません。

- KRKR / ONS / ARTEMIS（Android）：まず Tyranor 最新版を使用  
  [Tyranor 2.3.3 APK](https://t.shionlib.com/tools/Tyranor%202.3.3.APK)
- KRKR / ONS（iOS）：App Store で対応エミュレーターを各自インストール。
- OTHER：配布ページの注記に従って実行方法を選択。

最新版の Tyranor は、KRKR・ONS・Artemis の代表的なケースをカバーできます。配布ページに特定パッチや実行方法の指定がある場合は、そちらを優先してください。

## プラットフォーム別の推奨

### Windows

- まずネイティブ起動を優先。
- 配布ページに特定環境やパッチ指定がある場合は、その手順を優先。

### Android

- まずリソースカードのシミュレーターエンジンラベル（KRKR / ONS / ARTEMIS / OTHER）を確認。
- KRKR / ONS / ARTEMIS：まず Tyranor 2.3.3 を使用。
- OTHER または非互換時：配布ページの注記に従って切り替える。

### iOS

- 現時点で Tyranor の iOS 版はありません。
- iOS ユーザーはラベルに合わせて App Store から KRKR / ONS エミュレーターをインストールしてください。
- ARTEMIS / OTHER ラベルの場合は、配布ページの互換性注記を優先してください。

## よくある質問

### 起動はするが黒画面/クラッシュする

まず同じエンジン向けの別バージョンのエミュレーターを試し、その後に配布ページの注記で特定パッチまたは特定バージョン指定がないか確認してください。

### エンジンが判別できない

まず配布ページの注記を確認します。記載がない場合は、フォルダ構成と起動ファイル名から判断し、それでも不明ならコメント欄/ディスカッションで配布者に確認してください。
