## PotatoVN とは

[PotatoVN](https://potatovn.net) はオープンソースの Windows 向けビジュアルノベル / ギャルゲー管理ツールです。ローカルゲームの自動検出、Bangumi/VNDB からのゲーム情報取得、プレイ時間・状態の記録、クラウドセーブ同期などの機能をサポートしています。

PotatoVN アカウントを Shionlib に連携すると、Shionlib のゲームページで PotatoVN のプレイデータを直接確認でき、Shionlib からゲームを PotatoVN のライブラリに追加することもできます。

## アカウントを連携する

**アカウント設定 → 連携アカウント** に移動し、**アカウントを接続** をクリックして、ダイアログに PotatoVN のユーザー名とパスワードを入力します。

> パスワードは PotatoVN サーバーからアクセストークンを取得するためのみに使用されます。Shionlib には保存されません。

連携が成功すると、ページに接続中のアカウント名とトークンの有効期限が表示されます。

## ゲームページでの使い方

アカウントを連携すると、ゲームエントリーページに **PotatoVN** カードが表示され、以下の機能が提供されます：

### ライブラリに追加

そのゲームがまだ PotatoVN ライブラリに追加されていない場合、**PotatoVN に追加** をクリックするとレコードを直接作成できます。

### プレイデータを確認

ライブラリに登録済みのゲームには、以下の同期データが表示されます：

| フィールド       | 説明                                          |
| ---------------- | --------------------------------------------- |
| 総プレイ時間     | 累計プレイ時間                                |
| 最終プレイ       | 最後のプレイセッションの日時                  |
| プレイ状態       | 予定 / プレイ済み / プレイ中 / 保留中 / 断念  |
| マイレーティング | PotatoVN で付けた評価（評価がある場合に表示） |

データは PotatoVN から同期されます。システムは毎時間同期タスクを実行し、最後の同期時刻がカードの下部に表示されます。

### ライブラリから削除

カード右上の **···** メニューをクリックして **PotatoVN から削除** を選択すると、PotatoVN クラウド上のそのゲームのすべてのデータ（プレイ時間と評価を含む）が削除されます。

> この操作は PotatoVN 上のゲームデータを完全に削除します。元に戻すことはできません。慎重に操作してください。

## トークンの自動更新

PotatoVN のアクセストークンには有効期限があります。Shionlib バックエンドは 6 時間ごとに自動的に確認し、**有効期限が 3 日未満のトークンを自動更新します**。通常、手動での対応は不要です。

現在のトークンの有効期限は **アカウント設定 → 連携アカウント** ページで確認できます：

- 日付が**オレンジ色**で表示されている場合、トークンは 7 日以内に期限切れになります（自動更新は残り 3 日未満でトリガーされます）
- 日付が**赤色**で「トークン期限切れ」と表示されている場合、自動更新が失敗したかトークンが完全に無効になっています

トークンが期限切れと表示された場合は、**接続を解除** をクリックしてから再度アカウントを連携して手動で回復してください。

## アカウントの連携解除

**アカウント設定 → 連携アカウント** に移動し、**接続を解除** をクリックすると、Shionlib と PotatoVN の連携が解除されます。

連携解除によりアカウントの関連付けのみが削除されます。PotatoVN 上のデータは削除されません。
