## 个人信息

### 修改用户名

在 **个人信息** 页面，点击用户名输入框修改显示名称，最多 20 个字符。修改后点击 **更新** 保存。

### 修改头像

点击当前头像，选择本地图片后会弹出裁剪工具。裁剪完成后点击 **更新头像** 按钮保存。

## 安全

### 更改密码

在 **安全** 页面填写当前密码和新密码。新密码要求：

- 至少 8 个字符
- 包含至少一个小写字母、一个大写字母和一个数字
- 不能包含空格

提交成功后，系统会自动退出登录，需重新登录。如果忘记当前密码，可以点击 **忘记密码** 按钮进行密码重置。

### 更改邮箱

更改邮箱需要两步验证：

1. 系统向当前邮箱发送验证码，填入后进入下一步
2. 填写新邮箱地址，获取并填入发送到新邮箱的验证码，完成更改

更改成功后同样会自动退出登录。

### 通行密钥（Passkey）

通行密钥是一种无需密码的登录方式，使用设备的生物识别（指纹、面部识别）或 PIN 码完成验证。

**添加通行密钥：**

1. 点击 **添加通行密钥**，可选填一个名称（最多 128 字符）便于识别
2. 浏览器弹出验证请求，按提示完成生物识别或 PIN 验证
3. 验证成功后，通行密钥自动保存并出现在列表中

**管理已有通行密钥：**

列表中每条通行密钥会显示名称、设备类型（可同步 / 单设备 / 未知）、是否已备份，以及最后使用时间。点击删除按钮可以移除对应通行密钥，确认后生效。

> 如果你的浏览器或设备不支持 WebAuthn，该功能将不可用。

## 站点设置

### NSFW 内容显示

控制成人内容在页面上的显示方式，有三个选项：

- **不显示** — 过滤所有 NSFW 内容
- **Spoiler 包裹** — 显示但默认遮罩，需手动展开
- **直接显示** — 不加遮罩展示所有内容

修改后需重新登录才能立即生效，或等待约 30 分钟后自动应用。

### 语言

选择界面语言（简体中文、English、日本語），保存后页面会自动切换至所选语言。

### 通知位置

设置页面右下角或其他位置显示操作通知的弹窗位置，共六个方位可选。此设置仅保存在本地浏览器中，不同步到账号。

## 下载设置

配置 Aria2 下载客户端，用于将游戏资源推送到本地或者远程运行的 Aria2 实例。

| 字段     | 说明                            |
| -------- | ------------------------------- |
| 协议     | HTTP 或 HTTPS                   |
| 主机     | Aria2 所在地址，如 `localhost`  |
| 端口     | 1 ~ 65535                       |
| 路径     | 如 `/jsonrpc`                   |
| 认证密钥 | Aria2 的 RPC Secret（如有设置） |
| 下载目录 | 留空则使用 Aria2 默认目录       |

填写完成后，可以点击 **测试连接** 验证配置是否正确。点击 **复位** 可将所有字段恢复为默认值。

此设置仅保存在本地浏览器中，不同步到账号。
