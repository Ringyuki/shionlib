## 打开下载面板

在任意游戏条目页面，点击页面上方的 **下载** 按钮，即可打开下载面板。

面板中会列出该游戏的所有可用资源，并按下载次数从高到低排列。

> 所有资源均由社区用户上传，请自行鉴别文件安全性。

## 读懂资源卡片上的标注

每张资源卡片的顶部都会显示若干标签，用于描述该资源的适用范围：

### 平台

标注该资源适用的运行平台，例如 **Windows**、**Android**、**iOS**、**PlayStation Vita** 等。

一张卡片可能同时标注多个平台，代表该压缩包内包含多平台版本，或该资源可跨平台运行。

### 语言

标注游戏/补丁所使用的语言，包括：

| 标注     | 含义                   |
| -------- | ---------------------- |
| 日本語   | 日语（原版或日语补丁） |
| 简体中文 | 简体中文汉化           |
| 繁體中文 | 繁体中文汉化           |
| English  | 英语版本或英语汉化     |
| 其它     | 其他语言               |

### 模拟器引擎（仅 Android / iOS）

仅在平台为 Android 或 iOS 时显示，标注该资源所使用的游戏引擎，用于帮助你选择合适的模拟器：

| 标注            | 引擎                     |
| --------------- | ------------------------ |
| KiriKiri (KRKR) | KiriKiri/KRKRZ 引擎      |
| ONScripter      | ONScripter 引擎          |
| Artemis         | Artemis Engine           |
| 其他引擎        | 其他引擎，请参考资源备注 |

关于各引擎对应的模拟器选择，请参阅 [常见引擎与模拟器对照](/zh/docs/guides/common-vn-engines-and-emulators)。

### 备注

部分资源会附带上传者填写的备注（显示在文件条目下方），说明该资源的版本信息、包含内容或特殊注意事项，建议下载前阅读。

## 文件条目说明

每张资源卡片下可能包含一个或多个文件条目，每个条目显示：

- **文件名与大小** — 确认你下载的是正确的文件
- **文件状态** — 见下方说明
- **哈希值** — 用于下载后校验文件完整性

### 文件状态

| 状态           | 含义                               |
| -------------- | ---------------------------------- |
| S3（绿色）     | 文件已就绪，可以下载               |
| 处理中（黄色） | 文件正在扫描和上传，请稍后刷新     |
| 等待上传       | 文件在队列中等待处理               |
| 仅自己可见     | 文件尚未完成处理，仅上传者本人可见 |

只有状态为 **S3** 的文件才能下载。

## 下载文件

点击文件条目右侧的按钮开始下载。下载前，系统会弹出一个 **Cloudflare Turnstile** 验证框（机器人验证），通过后才会生成下载链接。

> 下载链接有时效限制：10 GB 以内的文件有效期为 **120 分钟**，10~20 GB 的文件为 **240 分钟**，超过 20 GB 的文件为 **360 分钟**。请在有效期内完成下载。

### 普通下载

点击 **地球图标** 按钮，通过浏览器直接下载，适合大多数用户。

### 推送到 Aria2

点击 **闪电图标** 按钮，将下载链接推送到本地或远程运行的 Aria2 下载管理器。

使用此功能前，需先在 **账号设置 → 下载设置** 中配置 Aria2 的连接信息（协议、地址、端口、密钥、下载路径）。如果连接失败，系统会提示错误并引导你前往设置页。

## 校验文件哈希

下载完成后，建议对比文件的哈希值以确认文件完整性：

1. 复制文件条目中显示的哈希值（点击复制按钮）
2. 使用系统工具或第三方软件计算下载文件的哈希值
3. 对比两个值，完全一致则说明文件完整

Shionlib 目前支持 **BLAKE3** 和 **SHA-256** 两种哈希算法，文件条目中会明确标注使用的算法。

## 误报病毒标记

如果文件条目显示红色的 **「可能存在病毒」** 标签，说明自动扫描流程曾检测到疑似病毒特征，但管理员审核后认定为**误报**并手动放行。

标签的悬停提示中会显示扫描检测到的病毒名称，供你参考。

> 视觉小说游戏（尤其是使用 KiriKiri 等引擎的作品）因为打包方式特殊，**极易被杀毒软件误报**。在确认来源可信的情况下，可以放心下载。

## 更新历史

点击资源卡片右下角的 **···** 菜单，选择 **更新历史**，可以查看该资源的所有历史版本，包括文件大小、哈希值、更新时间及上传者填写的更新原因。

## 举报资源

如果你发现某个资源存在违规内容（如恶意软件、版权问题等），可以点击 **···** 菜单，选择 **报告违规**，并填写举报理由。恶意举报会受到相应处置。
