## 什么是 PotatoVN

[PotatoVN](https://potatovn.net) 是一款开源的 Windows 视觉小说/Galgame 管理工具，支持自动识别本地游戏、从 Bangumi/VNDB 获取游戏信息、记录游玩时长与状态、以及云端存档同步等功能。

将 PotatoVN 账号与 Shionlib 绑定后，你可以在 Shionlib 的游戏页面直接查看来自 PotatoVN 的游玩数据，也可以在 Shionlib 上将游戏添加到 PotatoVN 的游戏库中。

## 绑定账号

前往 **账号设置 → 账号关联**，点击 **连接账号**，在弹窗中输入你的 PotatoVN 账号名和密码。

> 密码仅用于向 PotatoVN 服务器请求访问令牌，不会被 Shionlib 存储。

绑定成功后，页面会显示当前绑定的账号名以及令牌的有效期。

## 在游戏页面使用

绑定账号后，游戏条目页面会显示 **PotatoVN** 卡片，提供以下功能：

### 添加到库

如果该游戏尚未添加到你的 PotatoVN 库，点击 **添加到 PotatoVN** 即可直接创建记录。

### 查看游玩数据

已在库中的游戏会显示以下同步数据：

| 字段       | 说明                                     |
| ---------- | ---------------------------------------- |
| 总游戏时间 | 累计游玩时长                             |
| 最近游玩   | 上次游玩的时间                           |
| 游玩状态   | 想玩 / 玩过 / 在玩 / 搁置 / 抛弃         |
| 我的评分   | 在 PotatoVN 上给出的评分（有评分时显示） |

数据从 PotatoVN 同步，系统每小时执行一次同步任务，卡片底部会显示最后一次同步的时间。

### 从库中移除

点击卡片右上角的 **···** 菜单，选择 **从 PotatoVN 移除**，可以删除该游戏在 PotatoVN 云端的全部数据，包括游玩时长和评分。

> 此操作将永久删除 PotatoVN 上的游戏数据，无法恢复，请谨慎操作。

## 令牌自动续期

PotatoVN 的访问令牌有有效期。Shionlib 后端会每 6 小时自动检查一次，**当令牌有效期不足 3 天时，系统会自动续期**，正常情况下无需手动处理。

在 **账号设置 → 账号关联** 页面可以查看当前令牌的到期日期：

- 日期以**橙色**显示时，表示令牌即将在 7 天内过期（自动续期将在有效期剩余 3 天内触发）
- 日期以**红色**显示并标注"令牌已过期"时，表示自动续期失败或令牌已彻底失效

如果令牌显示已过期，需要先点击 **断开连接**，再重新绑定账号以手动恢复。

## 解绑账号

前往 **账号设置 → 账号关联**，点击 **断开连接**，即可解除 Shionlib 与 PotatoVN 的绑定关系。

解绑只会移除账号关联，不会删除你在 PotatoVN 上的任何数据。
