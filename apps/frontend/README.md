# Shionlib Frontend

This is the frontend app of Shionlib, built with Next.js and TypeScript.
It provides the web interface for browsing and managing visual novel resources.

## Tech Stack

- Next.js
- React
- TypeScript
- pnpm

## Getting Started

```bash
pnpm install
pnpm dev
```

The app will run in development mode with hot reload.

## Common Commands

```bash
pnpm dev          # Start development server
pnpm build        # Build for production
pnpm start        # Start production server
pnpm lint         # Run ESLint
pnpm typecheck    # Run TypeScript checks
pnpm test:e2e     # Run Playwright E2E tests
```
