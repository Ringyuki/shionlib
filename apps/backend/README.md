# Shionlib Backend

This is the backend service of Shionlib, built with NestJS and TypeScript.
It provides APIs, background jobs, and core business logic for the platform.

## Tech Stack

- NestJS
- TypeScript
- Prisma
- pnpm

## Getting Started

```bash
pnpm install
pnpm start:dev
```

The server runs in watch mode for local development.

## Common Commands

```bash
pnpm start         # Start server
pnpm start:dev     # Start in watch mode
pnpm start:prod    # Start compiled app
pnpm build         # Build project
pnpm lint:check    # Run ESLint checks
pnpm typecheck     # Run TypeScript checks
pnpm test          # Run unit tests
pnpm test:api:e2e  # Run API e2e tests
```
